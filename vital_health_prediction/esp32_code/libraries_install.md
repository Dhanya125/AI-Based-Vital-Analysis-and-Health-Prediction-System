# Required Libraries for ESP32

Install these libraries in Arduino IDE:

## 1. MAX30105 Library
- Open Arduino IDE
- Go to Tools → Manage Libraries
- Search for "MAX30105"
- Install "MAX30105" by SparkFun

## 2. Adafruit MLX90614 Library
- Search for "Adafruit MLX90614"
- Install "Adafruit MLX90614 Library"

## 3. ArduinoJson Library
- Search for "ArduinoJson"
- Install "ArduinoJson" by Benoit Blanchon

## Alternative Installation (Manual)
If you prefer manual installation:

1. Download libraries from GitHub:
   - MAX30105: https://github.com/sparkfun/SparkFun_MAX3010x_Sensor_Library
   - MLX90614: https://github.com/adafruit/Adafruit-MLX90614-Library
   - ArduinoJson: https://github.com/bblanchon/ArduinoJson

2. Extract to Arduino/libraries folder

3. Restart Arduino IDE
